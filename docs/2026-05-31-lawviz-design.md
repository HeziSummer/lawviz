# LawViz Design v1.0 Archive

> Deprecated archive. This file is not a current product source of truth.

The original v1.0 design contained outdated assumptions, including public report URLs, public sharing without the current access-control boundary, and earlier rollout sequencing. Do not use it to override v1.2 or the current private/internal MVP plan.

Current direction:

- LawViz is a real AI-assisted legal case visualization product for lawyers.
- First delivery is for self-use, trusted friends, and internal team use.
- The product must preserve the full workflow: conversational case intake, structured follow-up, plan confirmation, generated report HTML, PDF/PNG export, result retrieval, and usage/cost tracking.
- ICP, public production deployment, public acquisition, live public payment, and final commercial pricing are deferred.
- Current source of truth: `docs/2026-05-31-lawviz-design-v1.2-FINAL.md`, `HANDOFF.md`, `CLAUDE.md`.
