# LawViz Editorial Ink Style Pack

This is the fourth LawViz visual style pack.

It is a style template, not a case-type content template.

Chosen base style:

- Primary: Open Design `Guizang Editorial E-Ink Deck`

Purpose:

- define an editorial, e-ink, high-contrast A4 portrait report visual system;
- provide a magazine-like alternative to the structural, consulting, and guide styles;
- keep the same 26 component baseline as the other LawViz style packs;
- stay reusable across multiple case-type content templates.

Files:

- `visual-kit.json`: normalized LawViz visual configuration, component catalog, renderer constraints, and identity system.
- `style.css`: independent A4 portrait deck styling.
- `component-showroom.html`: style-template component catalog and composition stress test.
- `chart-showroom.html`: chart component catalog with visible text data for validation.

Current scope:

- page shell, editorial header/footer, print size, and overflow discipline;
- monochrome/e-ink typography, cards, tables, ledgers, decision components, and connector diagrams;
- chart components: bar, stacked bar, donut, waterfall, risk quadrant, score gauge, and calendar strip;
- identity system for individual lawyer, law firm team, and corporate-style firm;
- the same 26 component baseline used by the other approved style packs.

Out of scope:

- case-type page sequence;
- Agent interview flow;
- final content-template samples;
- production renderer integration.

Open `component-showroom.html` and `chart-showroom.html` directly in a browser. These are static local samples and do not affect the backend renderer yet.
